# Premium Conversion Playbook

## 1) Value Design

Free must stay useful. Premium must sell automation + insight + convenience.

### Free

1. Basic daily logs (feeding/sleep/diaper/symptom).
2. Vaccine records and reminders.
3. Basic growth chart.
4. Basic memory timeline.
5. 1 child profile.

### Premium

1. AI summaries and action plans.
2. Advanced triage and red-flag routing.
3. Chronic care automation (templates and triggers).
4. Doctor share links + advanced PDF + FHIR package.
5. Multi-child and advanced family permissions.
6. Unlimited memory media and auto monthly stories.
7. Advanced lab trend analytics.
8. Country switch and schedule remapping wizard.

## 2) Paywall Timing

1. Trigger after first value milestone:
   - 3+ logs in first day
   - first reminder completed
2. Trigger at premium feature intent:
   - user taps doctor share
   - user taps AI summary
3. Lifecycle nudges:
   - D3, D7, D14 personalized cards

## 3) Plan Structure

1. Monthly
2. Annual (default recommended)
3. Family annual

## 4) Core Conversion Experiments

1. Trial length: 3 vs 7 days.
2. Paywall hero message:
   - "Doctor-ready insights"
   - "Zero missed care"
3. Annual pricing framing:
   - monthly equivalent vs yearly total.
4. Intro offer:
   - first year discount by region.

## 5) KPI Targets

1. Trial start >= 8%
2. Trial to paid >= 40%
3. D35 download to paid >= 2%
4. D30 retained active families >= 30%

## 6) Anti-churn Layer

1. Weekly progress recap.
2. Missed reminder recovery flow.
3. Risk-of-churn model:
   - no logs for 5 days
   - no reminders completed for 10 days
4. Win-back offers by segment.
